# Binance Futures Trading Bot

This is a command-line interface (CLI) based trading bot for the Binance USDT-M Futures testnet. It supports placing market and limit orders, with robust validation and structured logging.

## Features

-   Place **Market** and **Limit** orders.
-   Supports both **BUY** and **SELL** sides.
-   Robust input validation for all parameters.
-   Structured logging to `bot.log` for all actions, successes, and errors.
-   Securely manages API keys using a `.env` file.

## Setup and Installation

### 1. Prerequisites

-   Python 3.8 or higher.
-   A Binance Testnet account with API credentials.
